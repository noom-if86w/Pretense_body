・ストーリー
　インターネットの海からとあるファイルを見つけ、どういう理由かはともあれ、あなたはそのファイルをダウンロード・インストールした。
　

・トーク傾向
　西洋魔術メイン、少しキリスト教、僅かにその他宗教系魔術、あとはファンタジー的なものとか日常的何かとか。
　またトーク内容が正しくない場合もありますのでご注意を。正しくないのを見つけたら作者にご連絡ください。

　魔術に関しては現実世界のものをベースにしてますが、多少ファンタジー的な魔法を交えたトークも存在しています。
